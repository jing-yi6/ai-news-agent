# 🤖 AI News Agent

每日 AI 资讯摘要生成器，自动从 X (Twitter) 获取 AI 相关推文，智能过滤分类，生成精美的 Markdown 日报。

## ✨ 功能特点

- **🐦 多源获取**: 支持获取关注列表或指定用户的推文
- **🎯 智能过滤**: 基于关键词的 AI 内容识别，自动排除无关推文
- **📂 自动分类**: 使用 LLM 将推文智能分类为模型发布、研究进展、产品更新、教程资源等
- **🤖 LLM 摘要**: 支持 OpenAI、兼容格式 API、Anthropic Claude 生成智能摘要
- **📝 Markdown 输出**: 生成美观的日报，包含目录、互动数据、原文链接
- **🔧 插件架构**: 支持自定义数据源和 LLM Provider，易于扩展
- **🚀 无需 API Key**: 使用 twscrape 抓取，无需申请 X API

## 📦 安装

### 方式一：使用 Conda（推荐）

#### 1. 克隆仓库

```bash
git clone https://github.com/jing-yi6/ai-news-agent.git
cd ai-news-agent
```

#### 2. 创建 Conda 环境

```bash
# 从 environment.yml 创建环境
conda env create -f environment.yml

# 激活环境
conda activate ai-news-agent
```

#### 3. 配置环境变量

```bash
cp .env.example .env
# 编辑 .env 文件，填入你的 X 账号信息
```

#### 4. 验证安装

```bash
python run.py --mock
```

## 🔑 配置说明

### X (Twitter) 账号配置（必需）

由于使用 twscrape 抓取推文，需要提供 X 账号信息：

#### 方式一：使用 Cookies（推荐）

从浏览器开发者工具复制 cookies：

```bash
X_COOKIES=auth_token=xxx;ct0=yyy
```

获取方法：
1. 登录 X (Twitter) 网页版
2. 打开浏览器开发者工具（F12）
3. 切换到 Application/Storage 标签
4. 找到 Cookies，复制 `auth_token` 和 `ct0` 的值

#### 方式二：使用用户名密码

```bash
X_USERNAME=your_twitter_username
X_PASSWORD=your_twitter_password
X_EMAIL=your_email@example.com
X_EMAIL_PASSWORD=your_email_password
```

### LLM 配置（可选，用于智能分类）

```bash
LLM_PROVIDER=openai          # openai 或 anthropic
LLM_API_KEY=your_api_key
LLM_BASE_URL=https://api.openai.com/v1
LLM_MODEL=gpt-3.5-turbo
```

## 🚀 使用方法

### 完整命令行参数

```bash
python run.py --help
```

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--users` | 指定用户名列表（不含 @） | - |
| `--user-id` | 指定用户 ID（用于获取关注列表） | - |
| `--mock` | 使用模拟数据测试 | False |
| `--datasource` | 数据源类型 (x, mock) | 从环境变量读取 |
| `--date` | 指定日期 (YYYY-MM-DD) | 当天 |
| `--output` | 输出目录 | `output` |
| `--max-following` | 最多获取关注用户数 | 30 |
| `--tweets-per-user` | 每个用户获取推文数 | 10 |
| `--min-engagement` | 最小互动数过滤 | 0 |
| `--use-llm` | 使用 LLM 智能分类 | False |

## 🎯 分类规则

推文会被自动分类到以下类别：

| 类别 | 说明 |
|------|------|
| **模型发布** | 新模型发布、模型版本更新、新功能上线 |
| **研究进展** | 论文发表、技术突破、benchmark 结果、学术研究 |
| **产品更新** | 产品功能更新、UI改进、新特性上线、Beta版本 |
| **教程资源** | 教程、指南、学习资源、文档、代码示例 |
| **行业动态** | 融资、投资、收购、合作、招聘、公司动态 |
| **其他资讯** | 不属于以上类别的AI相关内容 |

**分类方式**：
- 默认使用关键词匹配进行快速分类
- 启用 `--use-llm` 后使用 LLM 进行智能分类

## ⚠️ 注意事项

1. **账号安全**：使用 Cookies 方式更安全，不会暴露密码
2. **用户 ID**：获取关注列表需要目标用户的数字 ID，非用户名
3. **LLM 可选**：不配置 LLM 也能运行，会使用关键词分类
4. **速率限制**：twscrape 会自动处理速率限制，但频繁抓取可能导致账号暂时受限
